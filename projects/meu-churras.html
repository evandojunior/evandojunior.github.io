Meu Churras

Aplicativo para gerenciamento de churrascos.

Next.jsTypeScriptTailwindCSSPostgreSQLPrismaNextAuth

O Desafio

O desafio era criar uma plataforma simples onde usuários pudessem organizar churrascos, calcular a quantidade ideal de carnes e bebidas por convidado, além de dividir os custos de forma justa.

A Solução

Desenvolvi uma aplicação full stack com Next.js e Prisma, utilizando NextAuth para autenticação. A interface foi construída com TailwindCSS para garantir responsividade.
