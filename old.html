GuardianDoc

 Aplicação para gestão de empresa contábeis, cujo o objetivo principal é fornecer para os clientes destas a praticidade armazenar dos Documentos Físcais Eletrônicos que por lei devem ser guardados por no mínimo cinco anos.

Outra funcionalidade que se destaca é a geração de guias de tributos interestaduais cuja fui o responsável por realizar a implementação.

Nesse projeto participei desde o processo de prototipação ao de implementação.

Na fase de desenvolvimento utilizamos o framework Laravel para a API e para e pensando nos modelos atuais Flux, Redux e Vuex, optamos por utilizar o Vue.js no desenvolvimento da Single Page Application.

Laravel
